# BellaMart
A shop for your needs


#TO_RUN THIS PROGRAM

1. clone repo ```https://github.com/Charles2205/BellaMart.git```
2. run ```cd bellamart``` 
3. run ```yarn add```
4. create ``` .env```file using the the example file
5. run ```yarn run dev```(for mac/linux & windows)
